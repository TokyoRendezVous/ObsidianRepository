## 输入判断

### 检查输入，不重新输入

```c
if (scanf("%d", &n) != 1){
	// 格式错误
	return 1；
}
```

```c
if (n < 下限 || n > 上限){
	// 范围错误
	return 1；
}
```

### 重新输入

```c
while (1){
	printf("请输入一个1~6的数字: ");
	
	if(scanf("%d", &n) != 1){
		printf("输入格式错误，请重新输入。\n");
		
		while(getchar() != '\n'){
			continue;
		}
	}
	
	if(n < 1 || n > 6){
		printf("数值范围错误，请重新输入。\n");
		continue;
	}
	
	break;
}
```
